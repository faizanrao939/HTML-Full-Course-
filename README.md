Lesson 1 — HTML Document Structure

Goal: Understand <!DOCTYPE>, <html>, <head>, and <body>.

<!-- lesson1.html -->
<!doctype html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <title>Lesson 1 — Document Structure</title>
  </head>
  <body>
    <h1>Welcome to Lesson 1</h1>
    <p>This is a bare-bones HTML page.</p>
  </body>
</html>

Save as lesson1.html and open in your browser.
